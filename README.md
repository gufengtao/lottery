# 双色球生成器 Plugin

> 生成符合规则的双色球号码，自动避免与历史 300 期重复

## 版本信息

- **版本**: 1.0.0
- **作者**: Claude Code User
- **许可**: MIT

## 安装

本 plugin 已位于 `~/.claude/plugins/lottery/`，无需额外安装。

## 快速开始

### 生成双色球号码

```bash
/lottery
```

### 更新历史数据

```bash
/lottery-update
```

## 命令列表

| 命令 | 说明 |
|------|------|
| `/lottery` | 生成符合规则的双色球号码 |
| `/lottery-update` | 从中国福彩网同步最近 300 期历史数据 |

## 参数说明

### /lottery 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--last-red` | 上期红球号码（逗号分隔） | 自动从历史数据获取 |
| `--last-blue` | 上期蓝球号码 | 自动从历史数据获取 |
| `--repeat` | 重复上期红球数量 | 0-2 |
| `--consecutive` | 连续号码组数 | 1（2 个连续） |
| `--same-tail` | 同尾号组数 | 1（2 个同尾） |

## 使用示例

```bash
# 使用默认规则生成
/lottery

# 指定上期号码生成
/lottery --last-red 05,12,18,22,27,31 --last-blue 09

# 自定义规则
/lottery --repeat 3 --consecutive 2
```

## 生成规则

### 基本规则

- **红球**: 6 个，范围 1-33，不重复
- **蓝球**: 1 个，范围 1-16
- **历史数据**: 不能与最近 300 期重复

### 默认约束

- 0-2 个红球重复上期（随机）
- 1 组连续号码（如 12, 13）
- 1 组同尾号（如 03, 13）
- 蓝球不跟上期重复

### 统计规律参考

#### 红球常见特征

| 特征 | 常见范围 | 说明 |
|------|---------|------|
| 奇偶比 | 3:3 或 4:2 / 2:4 | 极端比例后易回归均衡 |
| 大小比 | 3:3 或 2:4 / 4:2 | 连续偏向一侧后易回调 |
| 三区比 | 2:2:2 ~ 3:2:1 ~ 1:3:2 | 某区超多后易修正 |
| 和值 | 80-130 (平均 105-110) | 极端值后易回归中值 |
| 连号 | 1-2 组 (≈60-70%) | 3 连以上较少 |
| 重号 | ≈40-50% | 有时连续有时断档 |
| 冷热号 | 热号延续，冷号回补 | 冷号可遗漏 15-30 期 |

#### 蓝球常见特征

| 特征 | 常见范围 | 说明 |
|------|---------|------|
| 奇偶 | 奇数≈55% | 波动较大 |
| 大小 | 大号略多 | 大偶→小奇常见回调 |
| 质合 | 合数稍多 | |
| 重复/同尾 | 偶尔 | 如 4→14→4 |
| 冷热 | 遗漏 10 期 + 回补概率上升 | 非必然 |

## 输出格式

```
双色球号码
═══════════════════
红球：03  05  12  13  14  28
蓝球：09
═══════════════════
规则验证:
  ✅ 重复上期：2 个 (05, 12)
  ✅ 连续号码：12, 13; 13, 14
  ✅ 同尾号：03, 13 (尾数 3); 05, 15 (尾数 5)
  ✅ 蓝球不跟上期重复
```

## 数据来源

- **网站**: 中国福彩网官方 (kaijiang.zhcw.com)
- **历史数据路径**: `./data/lottery-history.json` (插件目录内)
- **存储内容**: 最近 300 期开奖记录

## 技能导出

本 plugin 导出以下技能函数供其他命令使用：

| 函数 | 说明 |
|------|------|
| `generateLottery(options)` | 生成双色球号码 |
| `rand(min, max)` | 生成随机整数 |
| `pad(n)` | 数字补零 |
| `getTail(n)` | 获取尾数 |
| `hasConsecutive(balls, count)` | 检查连续号码 |
| `getSameTailGroups(balls, size)` | 检查同尾号 |
| `countRepeat(balls1, balls2)` | 计算重复数量 |
| `findConsecutivePairs(balls)` | 查找连续对 |
| `findSameTailGroups(balls)` | 查找同尾组合 |
