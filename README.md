# 双色球生成器 Plugin

> 生成符合规则的双色球号码，自动避免与历史 300 期重复

## 版本信息

- **版本**: 1.2.0
- **作者**: Claude Code User
- **许可**: MIT

## 安装

本 plugin 已位于 `~/.claude/plugins/lottery/`，无需额外安装。

## 快速开始

### 生成双色球号码

```bash
/lottery
```

### 更新历史数据

```bash
/lottery-update
```

## 命令列表

| 命令 | 说明 |
|------|------|
| `/lottery` | 生成符合规则的双色球号码 |
| `/lottery-update` | 从中国福彩网同步最近 300 期历史数据 |

## 参数说明

### /lottery 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--last-red` | 上期红球号码（逗号分隔） | 自动从历史数据获取 |
| `--last-blue` | 上期蓝球号码 | 自动从历史数据获取 |
| `--repeat` | 重复上期红球数量 | 0-3（随机，权重 23/48/24/5） |
| `--consecutive` | 连续号码组数 | 0-2 组（随机，权重 30/68/2） |
| `--same-tail` | 同尾号组数 | 0-2 组（随机，权重 26/52/22） |
| `--skip-one` | 隔一号组数（差值=2） | 0-2 组（随机，权重 43/44/13） |

## 使用示例

```bash
# 使用默认规则生成
/lottery

# 指定上期号码生成
/lottery --last-red 05,12,18,22,27,31 --last-blue 09

# 自定义规则
/lottery --repeat 3 --consecutive 2

# 指定隔一号组数
/lottery --skip-one 1

# 组合参数
/lottery --repeat 1 --consecutive 1 --skip-one 1
```

## 生成规则

### 基本规则

- **红球**: 6 个，范围 1-33，不重复
- **蓝球**: 1 个，范围 1-16
- **历史数据**: 不能与最近 300 期重复

### 默认约束（改进版）

- **重复上期**：0-3 个红球重复上期（随机，0个23%/1个48%/2个24%/3个5%）
- **连续号码**：0-2 组连续号码（约 70% 概率出现 1 组）
- **同尾号**：0-2 组同尾号（约 74% 概率出现，3个同尾仅 6% 概率，过滤）
- **隔一号**：0-2 组差值=2 的号码对（约 57% 概率出现 1 组，如 31-33）
- **蓝球**：尽量不跟上期重复，考虑大小/奇偶交替规律（4 种策略随机加权）
- **前 20 期热度规则**：本期红球应全部在前 20 期内出现过（80% 概率优先选热号）

### 统计规律参考

#### 红球常见特征

| 特征 | 常见范围 | 说明 |
|------|---------|------|
| 奇偶比 | 3:3 或 4:2 / 2:4 | 极端比例后易回归均衡 |
| 大小比 | 3:3 或 2:4 / 4:2 | 连续偏向一侧后易回调 |
| 三区比 | 2:2:2 ~ 3:2:1 ~ 1:3:2 | 某区超多后易修正 |
| 和值 | 80-130 (平均 105-110) | 极端值后易回归中值 |
| 连号 | 1-2 组 (≈70%) | 3 连以上较少（约 5%） |
| **隔一号** | 1 组 (≈57%) | 差值=2 的号码对，如 31-33、08-10 |
| 重号 | ≈40-50% | 有时连续有时断档 |
| 同尾号 | 1 组 (≈74%) | 3 个同尾仅约 6% |
| 冷热号 | 热号延续，冷号回补 | 冷号可遗漏 15-30 期 |
| **前 20 期热度** | 全部在前 20 期出现 | 完全陌生的冷号极少同时开出多个 |
| **近10期热号** | 平均遗漏仅 4.4 期 | 开出的号码几乎全部来自近5-10期，冷号（近10期未出现）极少同期开出多个 |

#### 蓝球常见特征

| 特征 | 常见范围 | 说明 |
|------|---------|------|
| 奇偶 | 奇数≈55% | 波动较大 |
| 大小 | 大号略多 | 大偶→小奇常见回调 |
| **交替规律** | 约 60% 概率反向 | 大→小、小→大、奇→偶、偶→奇 |
| 质合 | 合数稍多 | |
| 重复/同尾 | 偶尔 | 如 4→14→4 |
| 冷热 | 遗漏 10 期 + 回补概率上升 | 非必然 |

## 号码博弈分析：反向策略（思维实验）

> 本节为纯统计博弈分析，探讨"什么号码彩民最少人买"，反向使用可指导选号避开冷僻区间。

### 核心原则：看起来合规，但让大多数人买错

#### 一、彩民密集号区间（高竞争）

| 彩民偏好 | 原因 | 对应号码区间 |
|:---|:---|:---|
| 生日号码 | 最常见选号思路 | **01-31**（尤其 01-12 月、01-31 日） |
| 热号追涨 | 近期高频出现 | 最近 10 期高频号 |
| 吉利数字 | 心理偏好 | 06、08、09、16、18、28 |
| 连号直觉 | 规律感 | 01-02、11-12、23-24 等明显连号 |
| 整数尾数 | 美观感 | 10、20、30 |

#### 二、最难中（低竞争）的号码特征

**红球策略：**
- 主选 **26-33 区间**：超出生日号范围，买的人极少
- 和值落在 **100-120**（正常范围，不引疑）
- 三区分布 **2:2:2**（最符合统计规律，掩盖冷僻性）
- 奇偶比 **3:3 或 4:2**（看起来均衡）
- 混入 **2 个冷号**（遗漏 15 期以上）作为掩护

**蓝球策略：**
- 避开 01-03、07-08（近期高频，彩民爱跟）
- 选 **中段偶数** 10、12、14（买的人不集中）
- 冷号回补：遗漏 20 期+ 的号，知道的人少但统计上合理

#### 三、示例构造

```
红球：05  19  26  29  31  32   蓝球：14
```

| 验证项 | 结果 | 说明 |
|:---|:---:|:---|
| 和值 | 142 | 偏高但不极端，不引疑 |
| 奇偶比 | 4:2 | 正常范围 |
| 大小比 | 2:4 | 大号偏多（冷僻） |
| 三区分布 | 1:1:4 | 三区重（冷僻） |
| 连号 | 31-32（1组） | 符合规律 |
| 蓝球14 | 遗漏 26 期 | 冷号，极少人选 |

**为什么难中：** 26、29、31、32 超出生日号范围；蓝球14 遗漏期长，知道的人少。

#### 四、反向应用：提高选号覆盖率

反向使用本分析，可以**避开冷僻区间**，向大众集中区靠拢：

```
跟大众热区 → 中奖后奖金分摊多，但对号概率高
选冷僻组合 → 万一中奖奖金多，但对号概率更低
```

> **结论**：理性选号应根据个人目标在"高中奖率"和"高奖金"之间权衡。

## 更新日志

### v1.3.0 - 近10期热号优化

- **近10期热号规则**：统计验证平均遗漏仅 4.4 期，开出号码几乎全部来自近5-10期，选号时排除"冷号"（近10期未出现的号码：01、05、13、26、29等）可大幅提升覆盖率
- **冷号定义更新**：近10期未出现的号码视为冷号，生成时自动降低权重
- **统计规律补充**：红球热度窗口从"前20期"收窄至"近10期"，更精准

### v1.2.0 - 隔一号与过滤优化

- **隔一号**：新增 `--skip-one` 参数，支持差值=2 的号码对（如 31-33、08-10），统计约 57% 概率出现 1 组
- **3 同尾过滤**：3 个号码同尾（如 08,18,28）历史概率仅 6%，新增 94% 概率重试过滤
- **权重精调**：重号/连号/同尾/隔一号权重均基于 397 期历史数据统计校准
- **蓝球策略**：4 种蓝球选择策略（交替/追热/遗漏回补/追趋势），更均衡分布

### v1.1.0 - 规则改进

- **连号优化**：改为 0-2 组随机（原固定 1 组），更符合实际开奖（约 60-70% 出现连号）
- **同尾号优化**：改为 0-1 组随机（原固定 1 组），约 50% 概率出现
- **热度规则**：新增前 20 期热号优先选择（80% 概率选择前 20 期出现过的号码）
- **蓝球优化**：新增大小/奇偶交替规律（约 60% 概率选择与上期相反的特征）
- **文档更新**：更新参数说明和规则描述
