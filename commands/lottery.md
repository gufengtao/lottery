# Lottery Command - 双色球生成器

生成符合特定规则的双色球号码，自动避免与历史 300 期重复。

## 参数说明

| 参数              | 说明           | 默认值      |
| --------------- | ------------ | -------- |
| `--last-red`    | 上期红球号码（逗号分隔） | 自动从历史数据获取 |
| `--last-blue`   | 上期蓝球号码       | 自动从历史数据获取 |
| `--repeat`      | 重复上期红球数量     | 0-2（随机） |
| `--consecutive` | 连续号码组数       | 1（2 个连续） |
| `--same-tail`   | 同尾号组数        | 1（2 个同尾） |

## 规则说明

1. **红球**：6 个，范围 1-33
2. **蓝球**：1 个，范围 1-16
3. **默认规则**：
   - 红球：
     - 0-2 个红球重复上期（随机）
     - 1 组连续号码（如 13, 14）
     - 1 组同尾号（如 03, 13）
   - 篮球：
     - 尽量不跟上期重复
4. **历史 300 期不能重复**

## 数据来源

- **数据来源**: 中国福彩网官方 (kaijiang.zhcw.com)
- **历史数据路径**: `../data/lottery-history.json` (插件目录内)
- **存储内容**: 最近 300 期开奖号码（包含期号、开奖日期、红球、蓝球）
- **自动清理**: 超出 300 期的记录自动删除
- **数据更新**: 使用 `/lottery-update` 命令从官网同步最新数据

## 相关命令

```bash
/lottery         # 生成新一期号码
/lottery-update  # 从官网更新历史开奖数据
```

## 输出格式

```
双色球号码
═══════════════════
红球：03  05  12  13  14  28
蓝球：09
═══════════════════
规则验证:
  ✅ 重复上期：05, 12
  ✅ 连续号码：13, 14
  ✅ 同尾号：03, 13 (尾数 3)
```

## 使用示例

```bash
# 使用默认规则生成
/lottery

# 指定上期号码生成
/lottery --last-red 05,12,18,22,27,31 --last-blue 09

# 自定义规则
/lottery --repeat 3 --consecutive 2
```

## 执行任务

请执行以下步骤生成双色球号码：

1. **解析参数**：从 `$ARGUMENTS` 中提取用户指定的参数
2. **读取历史记录**：从插件目录 `../data/lottery-history.json` 读取最近 300 期号码，获取最新一期作为上期号码
3. **调用生成器**：运行 plugin 中的生成器脚本
4. **输出结果**：按格式展示生成的号码和规则验证结果

## 实现代码

```bash
python <plugin_dir>/skills/lottery-generator.py $ARGUMENTS
```
